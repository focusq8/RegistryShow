# RegistryShow
RegistryShow is a Windows desktop application for viewing and managing Windows startup entries from the registry and Startup folders in one place.

Full description

RegistryShow is a C# and Windows Forms desktop tool that helps users inspect, monitor, and manage Windows startup items. It scans common startup-related registry locations such as Run, RunOnce, and Winlogon, along with user and system Startup folders, then displays the results in a searchable and sortable interface. The application also highlights modified Winlogon values, supports auto-refresh, lets users open entries directly in Regedit, open file locations, copy file paths, and remove unwanted startup entries.

Features

View startup entries from Windows registry keys and Startup folders
Monitor common startup locations including Run, RunOnce, and Winlogon
Highlight non-default Winlogon values
Search and sort startup items easily
Auto-refresh startup scans
Open registry paths directly in Regedit
Open the related file location in Explorer
Copy file paths
Delete startup entries and their associated files
